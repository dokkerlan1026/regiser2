# 憨吉聯合大醫院系統明文件（純前端版）

## 1. 系統定位
- 系統名稱：`憨吉聯合大醫院系統`
- 技術：`HTML + CSS + JavaScript`
- 部署方式：直接開啟 `web/index.html`
- 資料保存：瀏覽器 `localStorage`（鍵值：`hanji_hospital_system_v1`）

- 病患總覽

## 3. 可記錄事項（完整資料清單）

### 3.1 主檔（Master Data）
- **科別**：科別代碼、名稱
- **時段**：時段代碼、名稱
- **院區**：院區代碼、院區名稱（例如：中央院區、介仁院區、建國院區）
- **醫師**：醫師代碼、名稱、所屬科別
- **城市/行政區**：城市名稱、行政區清單
- **排班**：排班代碼、醫師、時段
- **規則排班**：星期、醫師、時段、限掛人數
- **給號設定**：起始號、預約限額、總限額、給號方式

### 3.2 病患資料
- 病患 ID（系統）
- 病歷號 `MedicalRecordNo`（流水）
- 身分證 / 居留證 `NationalId`
- 健保卡號 `HealthCardNo`
- 姓名、性別、生日（民國 7 碼）、年齡
- 電話
- 城市、行政區、詳細地址
- 建立時間、更新時間

### 3.3 掛號資料
- 掛號 ID
- 病患 ID
- 醫師 ID、科別 ID、時段 ID
- 院區 ID
- 掛號日期
- 掛號號碼
- 初診 / 複診
- 掛號狀態：`active / cancelled / transferred`
- 退掛原因、轉掛對應 ID
- 建立時間

### 3.4 醫囑資料
- 醫囑 ID
- 掛號 ID
- 醫囑類型（藥品、處置、治療、檢查、醫材、其他）
- 醫囑名稱
- 劑量、頻率、天數、總量
- 單價、小計
- 備註
- 建立時間

### 3.5 批價資料
- 批價 ID
- 掛號 ID
- 身份別（健保 / 自費）
- 折扣別（一般 / 員工 / 弱勢）
- 掛號費、診察費、部份負擔
- 醫囑原始金額、保險自付後醫囑金額
- 總計
- 收費時間、收費方式

### 3.6 公告資料
- 公告 ID
- 標題
- 內容
- 發布時間

### 3.7 系統事件紀錄（Event Log）
- 事件 ID
- 事件類型（掛號、退掛、轉掛、排班異動、批價等）
- 事件明細（JSON）
- 事件時間

## 4. 核心流程說明

### 4.1 掛號流程
1. 先輸入身分證查詢是否既有病患。
2. 若無資料，建立新病患並自動給病歷號（`001` 起）。
3. 選擇日期、科別、時段後，系統依排班過濾醫師。
4. 系統自動試算該醫師該時段該日下一號。
5. 送出後建立掛號紀錄並寫入事件紀錄。

### 4.2 退掛 / 轉掛流程
- 退掛：將掛號狀態改為 `cancelled`，保存退掛原因。
- 轉掛：原掛號標記 `transferred`，產生新掛號並串接來源/目標 ID。

### 4.3 醫囑流程
- 醫囑總量計算：`總量 = 劑量 × 頻率 × 天數`
- 小計：`總量 × 單價`
- 每筆醫囑與掛號關聯，方便後續批價與統計。

### 4.4 批價流程
1. 選擇掛號紀錄。
2. 帶入掛號費 / 診察費 / 部份負擔。
3. 彙總該掛號醫囑金額。
4. 依身份別計算自付金額，再依折扣別套用折扣。
5. 確認後產生批價紀錄。

### 4.5 刪除規則（管理安全）
- 科別刪除：若底下仍有醫師，不可刪除。
- 醫師刪除：若已有掛號歷史，不可刪除。
- 刪除醫師時，會同步移除其排班與規則排班資料。

### 4.6 正式列印樣板
- 掛號系統可列印「門診掛號單」：含病患、科別、醫師、時段、看診號碼。
- 批價系統可列印「門診收據」：含收據號、費用明細、折扣與總額。
- 列印採獨立視窗樣板，不會直接印整個操作頁面。

## 7. 使用方式
1. 直接開啟 `web/index.html`。
2. 進入 `web/data-management.html` 可進行個人資料修改。
3. 驗證方式為「身分證號 + 健保卡號」。
4. 初診掛號需填寫健保卡號。
